# :before 和 ::before 区别? 

> **伪元素由双冒号和伪元素名称组成**

?> **单冒号(:)用于CSS3伪类，双冒号(::)用于CSS3伪元素。**

> 双冒号是在当前规范中引入的，用于区分伪类和伪元素。但是伪类兼容现存样式，浏览器需要同时支持旧的伪类，比如:first-line、:first-letter、:before、:after等

> 对于CSS2之前已有的伪元素，比如:before，单冒号和双冒号的写法::before作用是一样的。
